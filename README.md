
# Project 600
Web development learning notes and case study



前几天在 YouTube 上偶然看到 whatsdev 主播关于如何学习 Web 前端/全栈的视频 —— [3 Steps to ACTUALLY Land Your FIRST Developer Job in 8 MONTHS](https://youtu.be/o75ZyxL2SME) ，感觉挺受启发的。

自从2018下半年开始学习 Web 前端开发以来，虽然前端领域各个知识模块基本上都学了一遍，但仍觉得自己当前的技能水平仍处于初级阶段，与前端人才市场的主流需求还有差距，有必要尽快以正确的方法来继续提升技能水平。

因此决定在接下来的时间里，争取平均每天约花3个小时、共200天/600个小时的时间，把前端领域主流的知识和技能模块从头再学一次，把自己的前端开发技能提升到市场主流需求的水平，为下半年实现职业转换的目标做好积累和准备。
