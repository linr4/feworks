
# Project 600
Web development learning notes and case study



偶然在 YouTube 上看到关于如何学习 Web 前端 / 全栈的视频 —— [3 Steps to ACTUALLY Land Your FIRST Developer Job in 8 MONTHS](https://youtu.be/o75ZyxL2SME) ，颇受启发。

自 2018 下半年开始学习 Web 前端开发以来，虽前端领域各个知识版块都学了一遍，但觉得自己的技能水平仍处于初级阶段，与前端人才市场的主流需求还有差距，有必要尽快以正确的方法来继续提升、更上层楼。

因此决定在接下来这一年间，争取平均每天约花 3 个小时、共 200 天 / 600 个小时的时间，把前端领域主流的知识和技能从头再学 / 练一次，把自己的前端开发技能提升到市场主流需求的水平，为后续实现职业转换的目标打好基础、做好准备。
