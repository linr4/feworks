### Algorithmic Challenges

#### 1. `for...of`  vs. `for...in` 

​    MDN 上的解释：

​	`for...in` iterates over property **names**, 

​	`for...of` iterates over property **values**:

```js
var arr = [3, 5, 7];
arr.foo = 'hello';

for (var i in arr) {
   console.log(i); // logs "0", "1", "2", "foo"
}

for (var i of arr) {
   console.log(i); // logs 3, 5, 7
}
```
